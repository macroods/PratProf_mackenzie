## TST06 Login com senha incorreta

**Caso de uso em que se baseia:** Login

**Cenário:** Fluxo principal

**Preparação:**

a) O usuário deve estar cadastrado no sistema, com os seguintes dados: Nome, email, senha.

**Passos para execução do teste:**

1. Acessar a página de login.
2. Digitar o email cadastrado corretamente e senha incorreta.

**Resultado esperado:**

a) O sistema exibe mensagem vermelha "Email ou senha incorretos".

**Resultado obtido:**

(x) Sucesso
( ) Não executado
( ) Falha
( ) Cancelado

**Descrição do resultado obtido:**

O sistema não permite o login com a senha incorreta.
