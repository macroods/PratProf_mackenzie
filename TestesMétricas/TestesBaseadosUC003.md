# Métricas dos Testes baseados no caso de uso "Cadastrar restaurante"

## Contagem de cenários e testes

| Contagem | Nome do teste  | Condições                                           |
| -------- | -------------- | --------------------------------------------------- |
| 1        | TST09          | Cadastro de restaurante bem‑sucedido (positivo)     |
| 2        | TST10          | Email do restaurante já cadastrado (negativo)       |
| 3        | TST11          | Nome do restaurante já cadastrado (negativo)        |
| 4        | TST12          | Campos obrigatórios vazios (negativo)               |


**$T_i$ (Quantidade de testes implementados) = 4**

Cobertura de Testes (implementados) = $\frac{T_i}{R_{ft}} = \frac{4}{4} = 1 = 100\%$

**$T_x$ (Quantidade de testes executados) = 4**

Cobertura de Testes (executados) = $\frac{T_x}{R_{ft}} = \frac{4}{4} = 1 = 100\%$

**$T_s$ (Quantidade de testes bem-sucedidos) = 3**

Cobertura de Testes (bem-sucedidos) = $\frac{T_s}{R_{ft}} = \frac{4}{3} = 0,75 = 75\%$
