## TST12 Campos obrigatórios vazios no cadastro de restaurante

**Caso de uso em que se baseia:** Cadastrar restaurante

**Cenário:** Campos obrigatórios em branco (negativo)

**Preparação:**

a) A página de cadastro está acessível.

**Passos para execução do teste:**

1. Acessar a página de cadastro de restaurante.
2. Deixar o campo Email ou Senha (ou Nome) vazio.
3. Clicar em "Cadastrar Restaurante".

**Resultado esperado:**

a) Exibir mensagem de erro indicando campos obrigatórios (ex.: "Preencha todos os campos").  
b) Nenhum registro deve ser inserido no banco.

**Resultado obtido:**

(X) Sucesso  
( ) Não executado  
( ) Falha  
( ) Cancelado
