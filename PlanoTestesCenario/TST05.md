## TST05 Reserva de horário já ocupado

**Caso de uso em que se baseia:** Reservar mesa

**Cenário:** Fluxo principal

**Preparação:**

a) O usuário deve estar cadastrado no sistema, com os seguintes dados: Nome, email, senha.
b) O horário desejado já está reservado por outro usuário.

**Passos para execução do teste:**

1. Acessar a página inicial.
2. Clicar em um horário marcado como indisponível (vermelho).

**Resultado esperado:**

a) O sistema não permite a reserva e o símbolo do mouse troca para o símbolo "indisponível". 

**Resultado obtido:**

(x) Sucesso
( ) Não executado
( ) Falha
( ) Cancelado

**Descrição do resultado obtido:**

O sistema não permite a reserva corretamente.
