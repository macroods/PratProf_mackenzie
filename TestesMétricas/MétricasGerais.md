# Métricas dos Testes

## Contagem de cenários e testes

**$T_i$ (Quantidade de testes implementados) = 12**

Cobertura de Testes (implementados) = $\frac{T_i}{R_{ft}} = \frac{12}{12} = 1 = 100\%$

**$T_x$ (Quantidade de testes executados) = 12**

Cobertura de Testes (executados) = $\frac{T_x}{R_{ft}} = \frac{12}{12} = 1 = 100\%$

**$T_s$ (Quantidade de testes bem-sucedidos) = 11**

Cobertura de Testes (bem-sucedidos) = $\frac{T_s}{R_{ft}} = \frac{11}{12} = 0,9166 = 91,666\%$
