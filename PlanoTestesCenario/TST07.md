## TST07 Cadastro com campos obrigatórios vazios

**Caso de uso em que se baseia:** Cadastro de cliente

**Cenário:** Fluxo principal

**Preparação:**

a) Nenhuma preparação exigida.

**Passos para execução do teste:**

1. Acessar a página de cadastro.
2. Clicar em "Cadastrar" sem preencher nome, email ou senha.

**Resultado esperado:**

a) O sistema foca no campo não preenchido e exibe a mensagem "Preencha este campo".

**Resultado obtido:**

(x) Sucesso
( ) Não executado
( ) Falha
( ) Cancelado

**Descrição do resultado obtido:**

O sistema não faz o cadastro caso um dos campos não esteja preenchido e exibe a mensagem corretamente.
